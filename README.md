# homebrew-tap

Homebrew formulae for [tjakobsson](https://github.com/tjakobsson)'s tools.

## uatu

[uatu](https://github.com/tjakobsson/uatu) — local docs-tree watcher with
Markdown/AsciiDoc preview, review-load score, and an embedded terminal.

```bash
brew install tjakobsson/tap/uatu
brew upgrade uatu
```

Works on macOS and Linux, arm64 and x64.

## How this tap is maintained

`Formula/uatu.rb` is autogenerated and pushed by
[uatu's release workflow](https://github.com/tjakobsson/uatu/blob/main/.github/workflows/release.yml)
on every release — versions and checksums come from each release's
`SHA256SUMS`. Don't edit the formula by hand; it will be overwritten by
the next release. Issues with the formula belong in the
[uatu issue tracker](https://github.com/tjakobsson/uatu/issues).
